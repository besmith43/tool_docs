# Tar

This is a unix archive tool.  It pretty simple to use but it is very similiar to zip files.



### Basic Usage

##### Creating a Tar

```bash
	tar -cvf output.tar.gz file1 file2 dir1 dir2...
```

##### Listing a Tar's contents

```bash
	tar -tvf output.tar.gz
```

##### Expanding a Tar

```bash
	tar -xvf output.tar.gz
```
